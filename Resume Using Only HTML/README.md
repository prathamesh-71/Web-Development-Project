# My Resume (HTML Only)

Welcome to my resume repository! This resume is built exclusively with **HTML and Markdown**, hosted directly within this README.md file to showcase fundamental web presentation skills without relying on JavaScript or external frameworks.

# HTML Resume Project

This project is a personal resume website developed using only HTML. The main purpose of this project is to demonstrate my understanding of basic web development concepts and to showcase my profile in a structured format.

# About the Project

This resume webpage includes the following sections:

Personal Information

Career Objective

Educational Details

Technical Skills

Projects

Contact Information

The entire webpage is created using pure HTML without using CSS, JavaScript, or any external frameworks.

# Purpose

The objective of this project is to:

Strengthen core HTML fundamentals

Practice semantic structuring of web pages

Build a simple and professional resume layout

Understand webpage formatting using only HTML

# Technologies Used

HTML5

# How to Run

Download or clone the repository.

Open the HTML file in any web browser.
